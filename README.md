# this thing

Yet another yaml to graphviz script.
There is also some automation for better living.

## usage

```
bundle
mv example_source.yml source.yml
rake
# edit opened file and preview changes
# optionally remove source.yml from .gitignore

```

## depends on

- graphviz
- tmux
- vim
- eog :honeybee:
